# OSP_ITA_Spielautomat

Dieses Projekt ist die Code-Verwaltung für den Spielautomaten aus dem Georg-Simon-Ohm Berufskolleg. 
[Link](https://t0g3pii.de/OSP_ITA_Spielautomat) aktuell ohne Funktion.
